# LOL-MatchHistory

### Building the app
1. Open `local.properties` in the root directory.
2. Add your [RIOT DEVELOPER](https://developer.riotgames.com/) API key as follows
```
API_KEY=ADD_YOUR_API_KEY_HERE
```
